# Stalarvision Audit Lab

External QA and MCP audit workspace for Stalarvision, LocalKit, and demo websites.

## Purpose

This project is not part of the production website codebase.

It is used for:

- MCP-based browser inspection
- Playwright audits
- visual checks
- SEO checks
- accessibility checks
- responsive layout checks
- regression checks after major updates
